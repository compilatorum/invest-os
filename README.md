## 🧪 10. Meta-Prompt Chain (Deep Research)

### Nível 0 — Contexto & Perfil
```
SYSTEM: Você é o Web3 Investment OS no modo Curador Arquitetonico.
INPUT: {ATIVO} {TESE} {PERFIL} {CAPITAL}
TASK: Mapa cognitivo em 3 camadas:
  1. Técnica: whitepaper, tokenomics, arquitetura
  2. Semiótica: qual narrativa o projeto vende?
  3. Ontológica: que problema de verdade resolve?
OUTPUT: JSON { tecnica, semiotica, ontologica, flags_iniciais }
```

### Nível 1 — Due Diligence Financeira
```
INPUT: {mapa_cognitivo_nivel_0}
TASK: Calcule NVT, MVRV, SOPR, Sharpe 90d, IL Break-even, Temperatura
ALERTA: NVT_zscore > 2.0 E mindshare_growth > 30% → HYPE
OUTPUT: JSON { metricas, score_financeiro, alertas, recomendacao }
```

### Nível 2 — Due Diligence Regenerativa (KAIROS)
```
INPUT: {resultado_nivel_1}
TASK: Avalie sob 8 Formas de Capital
BLOQUEIO: Shannon < 1.2 → RECUSAR; Gini > 0.5 → ALERTA CRITICO
OUTPUT: JSON { capitais_8d, rhi_estimado, nivel_arquetipo, recomendacao }
```

### Nível 3 — Análise Semiótica (Peirce)
```
INPUT: {resultado_nivel_2}
TASK: Tríade Signo-Objeto-Interpretante
ALERTA: Ruído > 0.6 → especulação dominante
OUTPUT: JSON { coerencia, ruido, fase_alquimica, interpretacoes, acao_recomendada }
```

### Nível 4 — Decisão Axiológica & Gate Humano
```
INPUT: {resultados_niveis_0_3}
TASK: Score composto 40% financeiro + 35% regenerativo + 25% semiótico
ACAO: COMPRAR / AGUARDAR / REBALANCEAR / EVITAR
GATE: se valor > limite_perfil → confirmacao humana obrigatoria
OUTPUT: { acao, tamanho_posicao, score_composto, narrativa, metadata_nft }
```

### Nível 5 — Reflexão & Aprendizado (RLHF)
```
INPUT: {resultado_acao} + {feedback_humano_pos_decisao}
TASK: Registre em decisions.org: previsão vs. realidade, delta capital, sugestão de melhoria
OUTPUT: { log_rlhf, delta_capital, prompt_improvement, level_up? }
```

| MVRV | Market Cap / Realized Cap | < 1.0 = oportunidade; > 3.5 = perigo |
