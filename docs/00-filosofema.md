# Filosofema Central

> "Investir não é escolher ativos; é aumentar a capacidade adaptativa de sistemas vivos através da alocação inteligente de capitais múltiplos."

## Três Teses Fundacionais

### 1. Capital é Plural
Não existe uma única forma de capital. Existem 8 (ou mais) dimensões: financeiro, vivo, material, social, intelectual, experiencial, cultural, espiritual. Um sistema que otimiza apenas uma dimensão (ex: retorno financeiro) é frágil.

### 2. O Sistema Deve Produzir Mais Capital do que Consome
A métrica central não é ROI, mas RHI (Regenerative Health Index): a capacidade do sistema de regenerar os múltiplos capitais que utiliza. Um investimento só é bom se o RHI resultante for maior que o RHI inicial.

### 3. Investidor como Ecologista de Capitais
O investidor não é um "alocador de recursos", mas um curador de ecossistemas de capital. Cada decisão de alocação deve considerar o impacto sistêmico em todas as 8 formas de capital.
